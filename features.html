Built for production from day one
Every scaffold output passes CI/CD ready checks — typed, linted, and containerised out of the box.
Sub-2-second scaffolding
Parallel dependency resolution via Bun ensures your project is ready before you finish reading the confirmation message.
Auth.js v5 integrated
Session management, OAuth providers, and database adapters wired up at scaffold time. Zero manual configuration.
Prisma DB Connect
Schema, client, and migration baseline included. Supports PostgreSQL, MySQL, SQLite, and MongoDB connection strings.
Tailwind CSS v4 native
First-class Tailwind v4 support with the new CSS-first configuration model. No PostCSS config, no plugin setup.
TypeScript strict mode
All templates ship with tsconfig strict enabled, path aliases configured, and typed environment variable schemas via t3-env.
Docker-ready output
Multi-stage Dockerfile and .dockerignore generated alongside the project, optimised for each runtime and deployment target.
ESLint + Prettier wired
Opinionated lint rules aligned with the Vercel style guide. Husky pre-commit hooks enforced at the git layer.
CI/CD pipeline included
GitHub Actions workflows for test, build, and deploy to Vercel or Railway. Environment-scoped secrets handled automatically.
Security defaults
Content Security Policy headers, CSRF protection middleware, rate limiting, and OWASP-aligned input sanitisation included by default.

App Router React 19 RC Turbopack Server Actions
The production default. Server components, nested layouts, and streaming out of the box. The template targets Next.js 15 with the App Router and stable Server Actions enabled by default.
